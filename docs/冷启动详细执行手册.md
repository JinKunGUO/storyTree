  ---
  第一阶段：0→100 用户（第1-4周）

  Week 1：准备弹药

  1. 创建 3-5 棵示范故事树

  自己（或邀请1-2个朋友）创作标杆内容：
  - 1棵悬疑推理类（"凶手到底是谁？3条线索3个结局"）
  - 1棵恋爱类（"表白还是错过？你来决定走向"）
  - 1棵奇幻冒险类（"勇者的选择：每一步都通向不同命运"）

  每棵至少 3-5 个分支节点，让新用户进来有东西可读。

  2. 准备小红书素材（10-15条笔记草稿）

  素材类型：
  类型A：产品展示（3条）
    - "发现一个宝藏网站，可以写多结局小说！"
    - "AI帮我10分钟写出一棵故事树，太上头了"
    - "和朋友一起写同一个故事的不同走向，绝了"

  类型B：创作过程（3条）
    - "用AI写了一个悬疑故事，3个结局你选哪个"
    - "记录我写互动小说的全过程（附教程）"
    - "第一次用故事树写小说，新手也能上手"

  类型C：读者视角（3条）
    - "这个互动小说让我读了3遍，每次结局不一样"
    - "终于找到能和朋友一起写故事的地方了"
    - "比橙光还方便的互动小说创作工具"

  类型D：干货教程（3条）
    - "如何用AI写出你的第一本互动小说（保姆级教程）"
    - "分支叙事怎么写？故事树结构拆解"
    - "5个技巧让你的多结局故事更抓人"

  3. 配图/封面准备

  - 截图故事树的分支可视化页面（最直观的差异化展示）
  - 录屏AI续写过程做成GIF
  - 做对比图："传统小说 vs 分支小说"

  Week 2：小红书首发

  每日发布节奏：
  - 每天 1-2 条笔记
  - 发布时间：晚 8-10 点（活跃高峰）
  - 前 5 条用个人号发（真实感），后续可开品牌号

  关键词布局（标题+正文必带）：
  核心词：互动小说、AI写小说、多结局小说、分支小说
  长尾词：AI写作工具推荐、和朋友一起写小说、故事树、
         像游戏一样的小说、选择不同结局不同
  竞品词：橙光平替、比橙光简单、不用画画的互动小说

  话题标签：
  #AI写作 #互动小说 #宝藏网站 #写作灵感 #小说推荐 #创作日常

  Week 3：扩圈触达

  1. 小红书评论区引流

  找到这些类型的热门笔记，在评论区自然互动：
  - "有没有好用的AI写作工具"
  - "想写互动小说但橙光好难"
  - "有没有能和朋友一起写小说的地方"
  - "推荐几个小众写作平台"

  不要硬广，用"我最近在用一个叫故事树的网站，可以写多结局的"这种语气。

  2. 写作类社群渗透

  ┌────────┬────────────────────────┬────────────────────────────┐
  │  平台  │        目标社群        │            方式            │
  ├────────┼────────────────────────┼────────────────────────────┤
  │ QQ     │ 写作交流群、同人创作群 │ 分享创作过程截图+邀请码    │
  ├────────┼────────────────────────┼────────────────────────────┤
  │ 微信   │ 写手互助群             │ 发起"协作写故事"挑战       │
  ├────────┼────────────────────────┼────────────────────────────┤
  │ 豆瓣   │ 创意写作小组           │ 发帖分享工具推荐           │
  ├────────┼────────────────────────┼────────────────────────────┤
  │ LOFTER │ 同人圈                 │ "来用故事树写你CP的多条线" │
  └────────┴────────────────────────┴────────────────────────────┘

  3. 首批用户激励

  对进入平台的前 100 名用户：
  - 赠送 500 积分（价值约 ¥45）
  - 开放 30 天会员体验
  - 拉入"创始用户"微信群，直接收集反馈

  Week 4：数据复盘 + 调整

  看哪些笔记数据好：
  - 哪种标题点击率高
  - 哪种内容形态互动多（评论/收藏/分享）
  - 用户从哪条笔记来的最多

  用户访谈（微信群内1v1聊）：
  - "你为什么来试用？"
  - "第一个觉得不爽的点是什么？"
  - "你会推荐给朋友吗？为什么/为什么不？"

  ---
  第二阶段：100→1000 用户（第5-12周）

  核心策略：让 100 个用户带来 900 个

  1. 用户自传播激励（产品内）

  当前邀请码机制：邀请人+25，被邀请人+50
  建议升级为阶梯奖励：
    - 邀请第1人：双方各+50积分
    - 邀请第3人：解锁"高级AI续写"功能7天
    - 邀请第5人：获得"创始用户"专属徽章
    - 邀请第10人：永久获得每月额外5次AI插图

  2. 小红书规模化内容

  从个人号扩展到矩阵：
  - 主号： 品牌号"故事树StoryTree"，发产品更新+精选故事
  - 创作者号： 邀请3-5个活跃用户分享自己的创作过程
  - 教程号： 专注"AI写作教程""互动小说怎么写"等搜索流量

  每周内容矩阵：
  周一：用户故事/案例（"她用故事树写了一本10万字的互动小说"）
  周二：AI创作教程（保姆级，可被收藏）
  周三：作品推荐（"本周最佳故事树推荐"）
  周四：互动内容（"这个分支你选A还是B？评论区决定"）
  周五：产品更新/幕后（建立信任感）
  周末：轻松内容（写作meme/创作者日常）

  3. B站长视频（辅助渠道）

  - 录制 5-10 分钟的完整创作演示视频
  - 标题："用AI写一本互动小说需要多久？| 故事树测评"
  - B站的搜索长尾流量持久，一条好视频能持续引流数月

  4. 微信公众号沉淀

  - 每周发布 1 篇"本周热门故事树"精选
  - 读者在公众号看到精彩片段 → 点击跳转网站继续阅读
  - 积累私域流量，不受平台算法波动影响

  5. 创作者挖掘计划

  主动联系小红书/LOFTER上的写作博主（粉丝 1000-5万的小博主性价比最高）：
  话术模板：
  "Hi！我是故事树的开发者，看到你写的[具体作品名]非常棒。
  故事树是一个AI辅助的分支小说创作平台，想邀请你来体验一下。
  可以赠送你3个月高级会员+1000积分，如果你喜欢也可以分享给粉丝。
  没有硬性要求，纯粹觉得这个工具和你的创作风格很搭。"

  目标：找到 10 个愿意持续使用并分享的创作者

  第三阶段：留存策略（贯穿全程）

  用户留下来的 5 个理由

  ┌──────────────┬────────────────────────────┬─────────────────┐

  ├──────────────┼────────────────────────────┼─────────────────┤
  │ 日常习惯     │ 每日签到+连续签到奖励      │ ✅ 已有         │
  ├──────────────┼────────────────────────────┼─────────────────┤
  │ 创作沉没成本 │ 写了几万字的故事树舍不得走 │ ✅ 天然         │
  ├──────────────┼────────────────────────────┼─────────────────┤
  │ 社交关系     │ 协作伙伴、粉丝关注         │ ✅ 已有         │
  ├──────────────┼────────────────────────────┼─────────────────┤
  │ 进度激励     │ 等级、徽章、里程碑         │ ✅ 已有         │
  ├──────────────┼────────────────────────────┼─────────────────┤
  │ 内容消费     │ 持续有新故事可读           │ ⚠️ 需靠内容供给 │
  └──────────────┴────────────────────────────┴─────────────────┘

  关键留存动作

  新用户首日体验（决定留存的关键24小时）：

  注册 → 欢迎弹窗（已有）→ 引导阅读一个示范故事树（选择分支，体验"哇"时刻）
       → 引导用AI创作第一个章节（降低到零门槛）→ 发布成功+获得首个积分奖励
       → 推荐关注2-3个活跃创作者 → 次日推送"你的故事有新读者了"

  周活跃用户维护：
  - 每周推送"你关注的故事更新了"
  - 协作故事有人添加新分支时即时通知（已有通知系统）
  - 周榜公布：你的故事本周获得XX次阅读

  流失用户召回：
  - 3天未登录：推送"你的AI配额即将刷新"
  - 7天未登录：推送"有人在你的故事下留了评论"
  - 14天未登录：发放50积分召回礼包

  总结： 小红书是种草属性最强的平台，且"AI写作""互动小说"在小红书上竞争还不激烈，现在入场能吃到搜索流量红利。核心打法就是：用视觉化的故事树截图+AI创作过程，在小红书持续产出
  内容，让"故事树"这个名字和"多结局互动小说"心智绑定。
