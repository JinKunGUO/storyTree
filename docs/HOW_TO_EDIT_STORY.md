# 📝 如何编辑已存在的故事内容

**更新时间**: 2026-02-28  
**系统**: StoryTree

---

## 📚 理解故事结构

在StoryTree中，一个故事由两部分组成：

### 1. 故事基本信息 (Story)
- 标题 (title)
- 简介 (description)
- 封面图片 (cover_image)
- 作者信息
- 创建时间

### 2. 故事内容 (Nodes/Chapters)
- 章节标题
- 章节正文内容
- 章节路径
- 创建时间

```
┌─────────────────────────────────┐
│  Story (故事)                   │
│  ├─ title: "神秘的古堡"         │
│  ├─ description: "一个冒险故事" │
│  └─ cover_image: "cover.jpg"    │
│                                  │
│  Chapters (章节内容)             │
│  ├─ Chapter 1: "开端"           │
│  │   └─ content: "很久以前..."  │
│  ├─ Chapter 2: "探险"           │
│  │   └─ content: "他们发现..."  │
│  └─ Chapter 3: "结局"           │
│      └─ content: "最终..."      │
└─────────────────────────────────┘
```

## ✅ 编辑故事基本信息

### 适用场景
- 修改故事标题
- 更新故事简介
- 更换故事封面

### 操作步骤

#### 1. 打开故事详情页

访问你的故事：
```
http://localhost:3001/story?id=7
```

#### 2. 找到编辑按钮

在故事信息区域，你会看到：
```
┌─────────────────────────────┐
│  故事标题                   │
│  故事简介...                │
│                             │
│  [开始阅读] [编辑故事] ✏️  │
└─────────────────────────────┘
```

**注意**：
- ✅ 只有故事作者能看到"编辑故事"按钮
- ❌ 其他用户看不到此按钮

#### 3. 打开编辑模态框

点击"编辑故事"按钮，会弹出编辑框：

```
┌──────────────────────────────┐
│  ✏️ 编辑故事信息        ×   │
├──────────────────────────────┤
│                              │
│  故事标题 *                  │
│  ┌────────────────────────┐ │
│  │ 神秘的古堡             │ │
│  └────────────────────────┘ │
│                              │
│  故事简介 *                  │
│  ┌────────────────────────┐ │
│  │ 一个关于冒险的故事...  │ │
│  │                        │ │
│  └────────────────────────┘ │
│                              │
│  故事封面                    │
│  ┌────────────────────────┐ │
│  │ [封面预览]             │ │
│  └────────────────────────┘ │
│  ┌────────────────────────┐ │
│  │ https://...            │ │
│  └────────────────────────┘ │
│                              │
│     [取消]  [保存修改] 💾   │
└──────────────────────────────┘
```

#### 4. 修改内容

- **标题**：直接修改文本（必填，最多100字符）
- **简介**：修改描述（必填，最多500字符）
- **封面**：输入图片URL，会实时预览

#### 5. 保存修改

点击"保存修改"按钮：
- ✅ 显示"保存成功！"
- ✅ 页面自动刷新
- ✅ 显示更新后的内容

## ✅ 编辑章节内容

### 适用场景
- 修改章节标题
- 编辑章节正文
- 修正错别字
- 添加新内容

### 操作步骤

#### 1. 打开章节阅读页

从故事详情页点击某个章节，或直接访问：
```
http://localhost:3001/chapter?id=1
```

#### 2. 找到编辑按钮

在页面右上角工具栏：
```
┌─────────────────────────────────┐
│  Chapter Title                  │
│                    [✏️] [⚙️] [🔖] │
│                     ↑            │
│                   编辑按钮        │
└─────────────────────────────────┘
```

**位置**：在"设置"和"书签"按钮之间

**注意**：
- ✅ 只有章节作者能看到编辑按钮
- ❌ 其他用户看不到此按钮

#### 3. 打开编辑模态框

点击 ✏️ 编辑按钮，会弹出编辑框：

```
┌──────────────────────────────────┐
│  ✏️ 编辑章节            ×       │
├──────────────────────────────────┤
│                                  │
│  章节标题 *                      │
│  ┌────────────────────────────┐ │
│  │ 第一章 - 开端              │ │
│  └────────────────────────────┘ │
│                                  │
│  章节内容 *                      │
│  ┌────────────────────────────┐ │
│  │ 很久很久以前，在一个遥远  │ │
│  │ 的王国里...                │ │
│  │                            │ │
│  │                            │ │
│  │                            │ │
│  │                            │ │
│  │                            │ │
│  └────────────────────────────┘ │
│  当前字数: 1250                  │
│                                  │
│     [取消]  [保存修改] 💾       │
└──────────────────────────────────┘
```

#### 4. 编辑内容

- **章节标题**：修改标题（必填，最多100字符）
- **章节内容**：编辑正文（必填，无字数限制）
- **实时字数统计**：底部显示当前字数

#### 5. 保存修改

点击"保存修改"按钮：
- ✅ 显示"保存成功！"
- ✅ 页面自动刷新
- ✅ 显示更新后的内容

## 🎯 完整编辑示例

### 场景：编辑故事《神秘的古堡》

假设你有一个故事ID为7，包含3个章节：

#### 步骤1：编辑故事基本信息

1. **访问故事详情页**：
   ```
   http://localhost:3001/story?id=7
   ```

2. **点击"编辑故事"**

3. **修改内容**：
   ```
   标题: 神秘的古堡探险记  (从"神秘的古堡"改为"神秘的古堡探险记")
   简介: 一个关于勇气与智慧的冒险故事，讲述了三个少年探索古堡的奇遇...
   封面: https://example.com/new-cover.jpg
   ```

4. **保存** → ✅ 故事信息已更新

#### 步骤2：编辑第一章内容

1. **在故事详情页点击"第一章"**，或访问：
   ```
   http://localhost:3001/chapter?id=1
   ```

2. **点击右上角的 ✏️ 编辑按钮**

3. **修改内容**：
   ```
   标题: 第一章 - 古堡的传说  (从"第一章 - 开端"改名)
   
   内容: 
   很久很久以前，在一个遥远的王国里，有一座神秘的古堡。
   
   传说这座古堡里藏着无数的宝藏和秘密，但也充满了危险...
   
   (继续编辑正文内容)
   ```

4. **保存** → ✅ 第一章已更新

#### 步骤3：编辑第二章内容

1. **返回故事详情页**，点击"第二章"

2. **点击 ✏️ 编辑按钮**

3. **修改内容**：
   ```
   标题: 第二章 - 初入古堡
   
   内容: 
   三个少年站在古堡的大门前，心中充满了期待和紧张...
   
   (编辑正文)
   ```

4. **保存** → ✅ 第二章已更新

#### 步骤4：编辑第三章内容

重复上述步骤编辑第三章。

## 🔐 权限说明

### 谁可以编辑？

#### 故事基本信息
- ✅ **故事作者**可以编辑
- ❌ 其他用户**不能**编辑

#### 章节内容
- ✅ **章节作者**可以编辑
- ❌ 其他用户**不能**编辑

**注意**：在StoryTree中，不同章节可能有不同的作者（协作创作）。

### 如何判断是否是作者？

系统会自动检查：
```javascript
// 检查当前用户是否是作者
if (story.authorId === currentUserId) {
    // 显示编辑按钮
} else {
    // 隐藏编辑按钮
}
```

如果你看不到编辑按钮：
- ❌ 你没有登录
- ❌ 你不是这个故事/章节的作者

## 🛠️ 技术细节

### API接口

#### 更新故事信息
```http
PUT /api/stories/:id
Authorization: Bearer <token>
Content-Type: application/json

{
  "title": "新标题",
  "description": "新简介",
  "coverImage": "https://..."
}
```

#### 更新章节内容
```http
PUT /api/nodes/:id
Authorization: Bearer <token>
Content-Type: application/json

{
  "title": "新章节标题",
  "content": "新章节内容"
}
```

### 前端文件

- **故事详情页**: `web/story.html`
  - 编辑故事基本信息的UI和逻辑

- **章节阅读页**: `web/chapter.html`
  - 编辑章节内容的UI和逻辑

### 后端路由

- **故事路由**: `api/src/routes/stories.ts`
  - `PUT /:id` - 更新故事信息

- **章节路由**: `api/src/routes/nodes.ts`
  - `PUT /:id` - 更新章节内容

## 🐛 常见问题

### Q1: 看不到"编辑故事"按钮？

**原因**：
- 你没有登录
- 你不是故事作者

**解决**：
1. 确保已登录：`http://localhost:3001/login`
2. 确保使用的是创建该故事的账号

### Q2: 看不到 ✏️ 编辑按钮？

**原因**：
- 你没有登录
- 你不是章节作者

**解决**：
1. 确保已登录
2. 确保使用的是创建该章节的账号

### Q3: 点击保存后出现500错误？

**原因**：
- API服务器未运行
- 数据库连接失败
- 字段验证失败

**解决**：
1. 检查API服务器：`curl http://localhost:3001/api/health`
2. 查看控制台错误信息
3. 确保所有必填字段都已填写

### Q4: 如何批量编辑多个章节？

**目前不支持批量编辑**，需要逐个编辑：
1. 打开章节1 → 编辑 → 保存
2. 打开章节2 → 编辑 → 保存
3. 打开章节3 → 编辑 → 保存

### Q5: 编辑后能恢复之前的版本吗？

**目前不支持版本历史**。建议：
- 在编辑前备份重要内容
- 或者在数据库层面做定期备份

## 🎯 总结

### 编辑故事 = 两步走

1. **编辑故事信息** (标题、简介、封面)

   - 点击"编辑故事"按钮

2. **编辑章节内容** (章节标题、正文)
   - 在章节阅读页
   - 点击 ✏️ 编辑按钮
