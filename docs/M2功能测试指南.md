## 🧪 功能测试清单

### 一、AI生成标记显示功能

#### 测试场景1：故事详情页章节列表

**测试步骤：**
1. 登录系统
2. 进入任意故事详情页（`/story?id=xxx`）
3. 查看章节列表

**预期结果：**
- ✅ AI生成的章节标题旁显示紫色渐变徽章
- ✅ 徽章显示"🤖 AI创作"文字
- ✅ 人工创作的章节不显示徽章
- ✅ 徽章样式美观，不影响标题阅读

**验证要点：**
```
- 徽章颜色：紫色渐变（#667eea → #764ba2）
- 徽章位置：章节标题右侧
- 响应式：移动端正常换行显示
```

#### 测试场景2：章节阅读页

**测试步骤：**
1. 点击一个AI生成的章节
2. 进入章节阅读页（`/chapter?id=xxx`）
3. 查看章节标题

**预期结果：**
- ✅ 章节标题旁显示AI徽章
- ✅ 徽章样式与列表页一致
- ✅ 不影响页面布局

**数据验证：**
```sql
-- 查看章节的AI标记
SELECT id, title, ai_generated FROM Node WHERE id = 'xxx';
```

1. 登录系统

- ✅ 显示当前积分数值

**验证要点：**

#### 测试场景2：导航栏等级显示

**测试步骤：**
1. 登录系统
2. 观察页面顶部导航栏的用户名区域

**预期结果：**

- ✅ 徽章小巧，不占用太多空间
- ✅ 在所有页面都显示

- ✅ 作者名旁显示等级徽章（如果已实现）

### 三、举报功能

#### 测试场景1：打开举报模态框

**测试步骤：**
1. 登录系统
2. 进入任意章节阅读页
3. 点击工具栏的"举报"按钮（旗帜图标）

**预期结果：**
- ✅ 弹出举报模态框
- ✅ 模态框包含以下元素：
  - 标题："举报章节"
  - 举报原因下拉选择框
  - 详细说明文本框（可选）
  - 字数统计（0/500）
  - "取消"和"提交举报"按钮

#### 测试场景2：未登录用户举报

**测试步骤：**
1. 退出登录
2. 进入章节阅读页
3. 点击"举报"按钮

**预期结果：**
- ✅ 提示"请先登录"
- ✅ 1.5秒后自动跳转到登录页
- ✅ 登录后返回当前页面

#### 测试场景3：提交举报

**测试步骤：**
1. 登录系统
2. 打开举报模态框
3. 选择举报原因（如"垃圾广告"）
4. 输入详细说明（可选）
5. 点击"提交举报"

**预期结果：**
- ✅ 按钮文字变为"提交中..."
- ✅ 按钮禁用，防止重复提交
- ✅ 提交成功后显示"举报成功！感谢您为社区环境做出贡献"
- ✅ 模态框自动关闭
- ✅ 按钮恢复正常状态

**验证举报原因选项：**
```
1. 垃圾广告
2. 不当内容
3. 暴力血腥
4. 色情低俗
5. 抄袭侵权
6. 其他
```

#### 测试场景4：表单验证

**测试步骤：**
1. 打开举报模态框
2. 不选择举报原因
3. 直接点击"提交举报"

**预期结果：**
- ✅ 显示错误提示"请选择举报原因"
- ✅ 焦点自动定位到原因选择框

## 🐛 已知问题排查

### 如果AI徽章不显示

**排查步骤：**
1. 检查章节数据中是否有`ai_generated`字段
2. 打开浏览器控制台，查看是否有JavaScript错误
3. 检查CSS是否正确加载

```javascript
// 在控制台检查章节数据
console.log(chapters); // 查看ai_generated字段
```

### 如果等级不显示

**排查步骤：**
1. 检查localStorage中的用户信息
```javascript
console.log(JSON.parse(localStorage.getItem('user')));
```
2. 确认API返回的用户信息包含level和points字段
3. 检查auth.js是否正确加载

### 如果举报功能报错

**排查步骤：**
1. 检查后端API `/api/nodes/:id/report` 是否已实现
2. 查看浏览器控制台的错误信息
3. 检查token是否有效

```javascript
// 在控制台检查token
console.log(localStorage.getItem('token') || sessionStorage.getItem('token'));
```

## 🚀 快速测试脚本

如果需要快速验证功能，可以在浏览器控制台运行以下脚本：

```javascript
// 快速测试AI徽章
console.log('=== AI徽章测试 ===');
const aiBadges = document.querySelectorAll('.ai-badge');
console.log(`找到 ${aiBadges.length} 个AI徽章`);

  console.log(`徽章${i+1}:`, badge.textContent, getComputedStyle(badge).background);

// 快速测试等级显示
console.log('\n=== 等级显示测试 ===');

levelBadges.forEach((badge, i) => {
  console.log(`徽章${i+1}:`, badge.textContent);
});

// 快速测试举报按钮
console.log('\n=== 举报功能测试 ===');
const reportBtn = document.getElementById('reportBtn');
if (reportBtn) {
  console.log('✅ 找到举报按钮');
  console.log('按钮可见:', reportBtn.offsetParent !== null);
} else {
  console.log('❌ 未找到举报按钮');
}

// 检查用户等级信息
console.log('\n=== 用户信息 ===');
const user = JSON.parse(localStorage.getItem('user') || '{}');
console.log('等级:', user.level || '未设置');
console.log('积分:', user.points || '未设置');
```
