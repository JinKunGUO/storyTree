**更新时间**: 2026-02-28  
**功能状态**: ✅ 已实现

---

## 🎯 快速开始

### 方法1: 从故事详情页续写（推荐）

1. **访问你的故事详情页**
   ```
   http://localhost:3001/story?id=7
   ```

2. **点击"续写章节"按钮**
   - 位置：在"开始阅读"和"编辑故事"按钮之间
   - 图标：✏️ 笔的图标
   - 颜色：紫色渐变

3. **自动跳转到写作页面**
   ```
   http://localhost:3001/write?storyId=7
   ```

4. **开始撰写**
   - 输入章节标题（例如：第二章 - 冒险开始）
   - 在富文本编辑器中撰写内容
   - 实时显示字数统计

5. **发布章节**
   - 点击右下角的"发布章节"按钮
   - 章节自动发布并显示在故事中

---

### 方法2: 直接访问写作页面

如果你知道故事ID，可以直接访问：
```

```

## 📋 详细操作步骤

### 步骤1: 打开故事详情页

访问你的故事：
```
http://localhost:3001/story?id=7
```

### 步骤2: 找到"续写章节"按钮

在故事信息区域，你会看到多个操作按钮：

```
┌─────────────────────────────────────────┐
│  📖 故事标题                            │
│  作者: jinhui                           │
│                                         │
│  故事简介...                            │
│                                         │
│  [开始阅读] [续写章节] ✏️ [编辑故事]   │
│              ↑                          │
│            点这里！                      │
└─────────────────────────────────────────┘
```

**按钮特征**：
- 📍 位置：在"开始阅读"和"编辑故事"之间
- 🎨 颜色：紫色渐变背景
- ✏️ 图标：笔的图标
- 📝 文字：续写章节

**注意**：
- ✅ 只有故事作者能看到此按钮
- ❌ 其他用户看不到此按钮
- ❌ 未登录用户看不到此按钮

### 步骤3: 进入写作页面

点击"续写章节"按钮后，会自动跳转到写作页面：

```
┌──────────────────────────────────────┐
│  📖 故事标题                         │
│  撰写第N章                           │
│                                      │
│  ┌────────────────────────────────┐ │
│  │ 输入章节标题...                │ │
│  └────────────────────────────────┘ │
│                                      │
│  ┌────────────────────────────────┐ │
│  │ [B] [I] [U] ... 工具栏         │ │
│  ├────────────────────────────────┤ │
│  │                                │ │
│  │ 开始撰写你的故事...            │ │
│  │                                │ │
│  │                                │ │
│  │                                │ │
│  └────────────────────────────────┘ │
│                                      │
│  字数: 0                             │
│                                      │
│  [AI续写建议] [保存草稿] [发布章节] │
└──────────────────────────────────────┘
```

### 步骤4: 撰写章节内容

#### A. 输入章节标题

在顶部的标题输入框中输入章节标题：

**示例**：
```
第二章 - 神秘的森林
第三章 - 意外的相遇
第四章 - 真相大白
```

**要求**：
- 必填字段
- 最多100个字符
- 建议格式：第X章 - 标题

#### B. 撰写章节正文

在富文本编辑器中撰写内容：

**编辑器功能**：
- 📝 **格式化工具栏**：
  - 标题（H1, H2, H3）
  - 粗体、斜体、下划线、删除线
  - 有序列表、无序列表
  - 缩进、对齐
  - 引用块、代码块
  - 文字颜色、背景色
  - 清除格式

- 📊 **实时字数统计**：
  - 左下角显示当前字数
  - 实时更新

- 🤖 **AI续写建议**（可选）：
  - 点击"AI续写建议"按钮
  - AI会根据当前内容生成3个续写方案
  - 可以选择使用或参考

**撰写技巧**：
```
很久很久以前，在一个遥远的王国里...

（第一段：设定场景）

主人公走进了神秘的森林，四周静悄悄的...

（第二段：展开情节）

突然，一个声音从树林深处传来...

（第三段：制造悬念）
```

### 步骤5: 保存或发布

#### 选项A: 保存草稿

如果还没写完，可以保存为草稿：

1. 点击右下角的 **"保存草稿"** 按钮（白色按钮）
2. 系统保存草稿
3. 可以稍后继续编辑

**草稿特点**：
- ✅ 已保存，不会丢失
- ❌ 其他用户看不到
- ✅ 可以随时编辑
- ✅ 可以随时发布

#### 选项B: 发布章节（推荐）

如果已经写完，可以直接发布：

1. 点击右下角的 **"发布章节"** 按钮（紫色渐变按钮）
2. 系统验证内容（标题和正文不能为空）
3. 发布成功后自动跳转到章节阅读页
4. 章节立即对所有用户可见

**发布要求**：
- ✅ 章节标题不能为空
- ✅ 章节内容不能为空
- ✅ 必须登录
- ✅ 必须是故事作者

### 步骤6: 查看发布的章节

发布成功后：

1. **自动跳转**到新章节的阅读页
2. **在故事详情页**可以看到新章节
3. **其他用户**可以阅读新章节

## 🔐 权限说明

### 谁可以续写章节？

- ✅ **故事作者**可以续写
- ❌ 其他用户**不能**续写
- ❌ 未登录用户看不到按钮

### 如何判断是否有权限？

系统会自动检查：
```javascript
if (currentUserId === story.authorId) {
    // 显示"续写章节"按钮
} else {
    // 隐藏按钮
}
```

### 如果看不到"续写章节"按钮？

**可能的原因**：
1. ❌ 你没有登录 → 请先登录
2. ❌ 你不是故事作者 → 只能作者续写
3. ❌ 浏览器缓存问题 → 刷新页面

**解决方法**：
1. 确保已登录：访问 `http://localhost:3001/login`
2. 确保使用的是创建该故事的账号
3. 刷新页面（Ctrl+R 或 Cmd+R）

## 🐛 常见问题

### Q1: 看不到"续写章节"按钮？

**原因**：
- 你没有登录
- 你不是故事作者

**解决**：
1. 确保已登录：`http://localhost:3001/login`
2. 确保使用的是创建该故事的账号
3. 刷新页面

### Q2: 点击"续写章节"后跳转到错误页面？

**原因**：
- 故事ID丢失
- URL参数错误

**解决**：
1. 检查URL是否包含 `?storyId=7`
2. 手动访问：`http://localhost:3001/write?storyId=7`

### Q3: 点击"发布章节"后没有反应？

**原因**：
- 标题或内容为空
- 网络请求失败
- API服务器未运行

**解决**：
1. 确保标题和内容都已填写
2. 检查浏览器控制台是否有错误
3. 确认API服务器正在运行：`curl http://localhost:3001/api/health`

### Q4: 发布后在故事详情页看不到新章节？

**原因**：
- 页面缓存
- 章节未成功发布

**解决**：
1. 刷新故事详情页（Ctrl+R 或 Cmd+R）
2. 检查是否显示"发布成功"消息
3. 查看浏览器控制台是否有错误

### Q5: 如何编辑已发布的章节？

**答案**：
1. 访问章节阅读页：`http://localhost:3001/chapter?id=章节ID`
2. 点击右上角的 ✏️ 编辑按钮
3. 修改内容后保存

详见：`docs/HOW_TO_EDIT_STORY.md`

### Q6: 可以删除已发布的章节吗？

**答案**：
目前系统**不支持删除章节**。建议：
- 在发布前仔细检查内容
- 如需删除，请联系管理员

### Q7: 如何查看我的所有草稿？

**答案**：
访问个人中心：`http://localhost:3001/profile`

在"我的作品"中可以看到所有草稿和已发布的章节。
