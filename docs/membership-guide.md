# 付费会员系统使用指南

## 🚀 快速开始

### 1. 数据库迁移
```bash
cd api
npx prisma migrate dev --name add_membership_system
npx prisma generate
```

### 2. 启动服务
```bash
npm start
```

会员 Worker 会自动启动，负责：
- 每日检查会员到期情况
- 发送到期提醒（7 天、3 天、1 天）
- 处理过期会员降级

### 3. 访问会员中心
访问：`http://localhost:3001/membership.html`

## 📋 会员套餐

| 套餐 | 价格 | 有效期 | AI 配额倍数 | 主要权益 |
|------|------|--------|-----------|----------|
| 体验会员 | ¥9.9 | 7 天 | 1.2x (Lv3+20%) | 会员徽章、响应优先、去广告 |
| 月度会员 | ¥39 | 30 天 | 1.5x (Lv4+50%) | 模型选择、高级编辑器、批量操作 |
| 季度会员 | ¥99 | 90 天 | 2.0x (Lv5+100%) | 所有月度权益 + 优先客服 |
| 年度会员 | ¥388 | 365 天 | 2.5x (Lv5+150%) | 所有季度权益 + 专属活动 |
| 企业版 | ¥999 | 365 天 | 无限 | 多账号管理、定制服务 |

**注意**: 体验会员每个用户仅限购买一次。

## 💡 配额计算示例

配额基于用户等级，会员提供倍数加成：

```
基础配额（Lv3 用户）:
- 续写：50 次/月
- 润色：30 次/月
- 插画：10 次/月

购买月度会员后（1.5x）:
- 续写：75 次/月
- 润色：45 次/月
- 插画：15 次/月
```

## 🛒 购买流程

1. **选择套餐** - 访问会员中心页面
2. **创建订单** - 点击"立即开通"
3. **支付** - 跳转支付页面（开发环境使用模拟支付）
4. **生效** - 支付成功后立即开通会员

### 开发环境测试支付
支付回调 URL：`/api/payment/callback/mock`
```bash
# 模拟支付成功
curl -X POST http://localhost:3001/api/payment/callback/mock \
  -H "Content-Type: application/json" \
  -d '{"orderId": "YOUR_ORDER_ID", "success": true}'
```

## 👑 管理后台

访问：`http://localhost:3001/admin.html`

### 会员管理功能
- 查看会员统计数据
- 会员列表查询（支持筛选、搜索）
- 订单列表查询
- 手动调整会员状态

### API 端点
```bash
# 获取会员统计
GET /api/admin/membership/stats

# 获取会员列表
GET /api/admin/membership/members?page=1&limit=20

# 获取订单列表
GET /api/admin/membership/orders?page=1&limit=20

# 手动调整会员状态
POST /api/admin/membership/members/:userId/adjust
```

## 🔔 到期提醒

系统自动在以下时间发送提醒：
- **7 天前**：提示会员即将到期
- **3 天前**：加强提醒
- **1 天前**：紧急提醒
- **到期后**：降级为免费用户，发送通知

所有通知通过站内消息系统发送。

## ⚠️ 注意事项

1. **体验会员限制**：每个用户仅限体验一次
2. **配额计算**：会员配额基于用户等级 × 会员倍数
3. **自动续费**：目前仅支持手动续费（自动续费功能待接入支付平台代扣）
4. **退款政策**：会员一经购买，不支持退款（特殊情况联系管理员）

## 🔧 常见问题

**Q: 会员到期后配额如何变化？**
A: 到期后立即降级为免费用户，配额恢复为对应等级的基础配额。

**Q: 会员期间升级了用户等级，配额会变吗？**
A: 会的。配额实时基于当前等级和会员等级计算。

**Q: 支持同时拥有多个会员吗？**
A: 不支持。新购买的会员会覆盖现有会员。

**Q: 会员可以转让吗？**
A: 会员与账号绑定，不支持转让。
