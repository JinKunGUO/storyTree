```

│                                                             │
```

## 🚀 快速开始

### 方式1：AI续写（小段落）

**步骤**：
1. 打开写作页面：`/write?storyId=1`
2. 撰写至少50字内容
3. 点击"AI续写"按钮
4. 选择风格（悬疑/温情/脑洞）
5. 等待生成（3-10秒）
6. 选择一个选项插入编辑器
7. 继续编辑或发布

**示例**：
```
你的内容：
"艾莉娅推开城门，一股冷风扑面而来..."

↓ 点击"AI续写" → 选择"悬疑"

AI插入：
"城内一片死寂，只有风吹过时发出的呜咽声。
她注意到地面上有一串奇怪的脚印..."

↓ 继续编辑

你的内容：
"艾莉娅推开城门，一股冷风扑面而来...
城内一片死寂，只有风吹过时发出的呜咽声。
她注意到地面上有一串奇怪的脚印...
她决定跟随这些脚印..."

↓ 发布章节
```

---

### 方式2：AI创作章节（整章）

**步骤**：
1. 打开故事详情页：`/story?id=1`
2. 在章节目录区域点击"AI创作章节"
3. 选择惊喜时间（立即/1小时后/今晚/明天）
4. 选择风格（悬疑/温情/脑洞）
5. 点击"开始生成"
6. 等待AI生成（3-10秒，或延迟执行）
7. 查看3个章节选项
8. 点击"接受并创建章节"
9. 自动创建新节点并刷新页面

**示例**：
```
故事已有章节：
- 第一章：勇者的启程
- 第二章：森林的试炼
- 第三章：神殿的秘密

↓ 点击"AI创作章节" → 选择"立即生成" + "悬疑"

AI生成3个选项：
1. 第四章 - 守护者的警告
2. 第四章 - 地下迷宫
3. 第四章 - 神秘的声音

↓ 选择"第四章 - 地下迷宫" → 点击"接受并创建章节"

自动创建新章节：
- 第四章：地下迷宫 [AI创作]
  内容：完整的500-1000字章节
  标记：ai_generated: true

↓ 页面刷新，显示新章节
```

## 🐛 常见问题

### Q1：AI创作章节需要先写内容吗？
**A**：❌ 不需要！AI会基于最近三章内容自动生成新章节。

### Q2：如果故事还没有章节怎么办？
**A**：AI会基于故事简介生成第一章。

### Q3：AI续写需要多少字？
**A**：小段落模式需要至少50字，整章模式无要求。

### Q4：延迟生成的任务在哪里查看？
**A**：访问 `/ai-tasks` 查看所有AI任务。

### Q5：可以编辑AI生成的章节吗？
**A**：可以！AI创建的章节与普通章节一样，可以在写作页面编辑。

### Q6：AI生成的章节会有标记吗？
**A**：是的，章节列表中会显示"AI创作"标记。

## 🎉 总结

**两种模式，满足不同需求：**

- 📝 **需要快速续写**？→ 使用"AI续写"（小段落）
- 📚 **需要完整章节**？→ 使用"AI创作章节"（整章）
- ⏰ **想要定时惊喜**？→ 使用"惊喜时间"功能
- 🎨 **想要特定风格**？→ 选择"悬疑/温情/脑洞"

**记住：**
- 小段落 = 写作页面 + 需要50字 + 插入编辑器
- 整章 = 章节目录 + 无需草稿 + 创建节点

## 🐛 常见问题

**Q: 为什么AI续写没有惊喜时间选项？**  
A: AI续写是快速补充内容，仅支持立即生成。如需惊喜时间，请使用"AI创作章节"。

**Q: 插入的内容可以修改吗？**  
A: 可以。AI续写插入编辑器后，你可以自由修改。

**Q: AI创作的章节可以编辑吗？**  
A: 可以。接受章节后会跳转到新章节页面，你可以继续编辑。

**Q: 如何查看延迟任务的结果？**  
A: 访问 http://localhost:3001/ai-tasks 查看任务列表。

**Q: 上下文"最近3章"是什么意思？**  
A: 从当前章节向上追溯，获取最近的3个章节作为AI生成的参考。
