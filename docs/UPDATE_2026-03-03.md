## 🎯 本次更新内容

### 1. 🔧 修复AI API认证问题

**问题**: 点击"AI续写建议"时返回401未授权错误

**原因**: AI路由使用错误的认证方式
- ❌ 期望: `x-user-id` header
- ✅ 实际: `Authorization: Bearer <JWT_TOKEN>`

**修复**: 
- 修改 `api/src/routes/ai.ts` 中的 `getUserId` 函数
- 改为使用标准JWT认证
- 从Authorization header中提取和验证token

**文件**: `api/src/routes/ai.ts`

---

### 2. ✨ 区分保存草稿和发布章节功能

**需求**: 用户希望区分两个按钮的行为
- 保存草稿: 保存后留在当前页面继续编辑
- 发布章节: 保存后自动跳转到章节详情页

**实现**:
```javascript
// 保存草稿 - 留在当前页面
async function saveChapter(shouldPublishAndRedirect = false) {
    if (shouldPublishAndRedirect) {
        // 发布模式: 跳转页面
        showSuccess('发布成功！即将跳转...');
        setTimeout(() => window.location.href = ..., 1500);
    } else {
        // 草稿模式: 留在当前页面
        showSuccess('草稿保存成功！');
        // 重新启用按钮，可继续编辑
    }
}
```

**新增功能**:
- 内容变更追踪（`hasUnsavedChanges`）
- 离开页面前的确认提示
- 保存后清除未保存标记

**文件**: `web/write.html`
