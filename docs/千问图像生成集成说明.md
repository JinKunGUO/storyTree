# 千问图像生成集成说明

## 📅 实现日期
2026-03-06

## 🎯 功能概述

将AI插图生成功能从OpenAI DALL-E 3切换到阿里云**千问通义万相**（Wanx）图像生成服务，实现国内可直接访问的AI插图生成功能。

## ✅ 实现内容

### 1. 千问图像生成API集成

#### API端点
- **提交任务**: `https://dashscope.aliyuncs.com/api/v1/services/aigc/text2image/image-synthesis`
- **查询状态**: `https://dashscope.aliyuncs.com/api/v1/tasks/{task_id}`

#### 支持的模型
- **wanx-v1** (默认) - 通义万相基础版
- **wanx-sketch-to-image-v1** - 草图转图片
- **wanx-style-repaint-v1** - 风格重绘

### 2. 核心功能实现

#### `callQwenImageAPI(prompt: string)` 函数

**功能**: 调用千问图像生成API，异步生成图片

**流程**:
1. 提交图像生成任务（异步模式）
2. 获取任务ID
3. 轮询任务状态（每2秒，最多60次）
4. 返回生成的图片URL

**代码示例**:
```typescript
async function callQwenImageAPI(prompt: string): Promise<{ imageUrl: string }> {
  const imageModel = process.env.QWEN_IMAGE_MODEL || 'wanx-v1';
  
  // 第一步：提交任务
  const submitResponse = await fetch(
    'https://dashscope.aliyuncs.com/api/v1/services/aigc/text2image/image-synthesis',
    {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${process.env.QWEN_API_KEY}`,
        'Content-Type': 'application/json',
        'X-DashScope-Async': 'enable' // 异步模式
      },
      body: JSON.stringify({
        model: imageModel,
        input: { prompt },
        parameters: {
          size: '1024*1024',
          n: 1
        }
      })
    }
  );
```

**特点**:


### 3. 任务状态管理

#### 状态流转
```
PENDING (待处理)
  ↓
RUNNING (生成中)
  ↓
SUCCEEDED (成功) / FAILED (失败)
```

#### 轮询机制
- **轮询间隔**: 2秒
- **最大次数**: 60次（2分钟）
- **超时处理**: 返回超时错误，提示用户重试

### 4. 错误处理

#### 错误类型
1. **API调用失败** - 网络错误、认证失败
2. **任务提交失败** - 参数错误、配额不足
3. **生成失败** - 内容违规、生成超时
4. **轮询超时** - 2分钟内未完成

#### 错误提示
```typescript
// 认证错误
throw new Error('千问API密钥无效，请检查QWEN_API_KEY');

// 配额不足
throw new Error('千问图像生成配额已用完');

// 内容违规
throw new Error('图像生成失败: 内容包含敏感信息');

// 超时
throw new Error('图像生成超时，请稍后重试');
```

## 🔧 配置说明

### 环境变量配置

在 `api/.env` 文件中添加：

```bash
# 千问API密钥（必需）
QWEN_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx

# 千问文本模型（可选，默认qwen-plus）
QWEN_MODEL=qwen-plus

# 千问图像模型（可选，默认wanx-v1）
QWEN_IMAGE_MODEL=wanx-v1
```

### 获取千问API密钥

1. 访问 [阿里云DashScope控制台](https://dashscope.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 开通DashScope服务
4. 创建API Key
5. 复制API Key到 `.env` 文件

### 模型选择

| 模型 | 说明 | 价格 | 推荐场景 |
|------|------|------|----------|
| **wanx-v1** | 通义万相基础版 | ¥0.05/张 | 通用插画生成 |
| wanx-sketch-to-image-v1 | 草图转图片 | ¥0.08/张 | 草图上色 |

## 💡 使用建议

### 1. 提示词优化

**好的提示词示例**:
```
古风庭院，月光下，一位白衣少年独自站在梅花树下，神情忧郁，
远处灯火阑珊，水墨画风格，细腻唯美，富有故事感
```

**要点**:
- 明确场景（古风庭院、月光）
- 描述人物（白衣少年、神情忧郁）
- 添加细节（梅花树、灯火）
- 指定风格（水墨画、细腻唯美）

### 2. 风格选择

| 小说类型 | 推荐风格 |
|---------|---------|
| 武侠、仙侠 | 水墨画风格、国风插画 |
| 现代都市 | 现代插画、写实风格 |
| 奇幻、玄幻 | 奇幻插画、魔幻风格 |
| 言情、青春 | 唯美插画、温馨风格 |

### 3. 避免内容审核

**避免的内容**:
- 暴力、血腥场景
- 政治敏感内容
- 版权人物（明星、动漫角色）
- 过度性感的描述

**安全的描述**:
- 使用"战斗"代替"血腥"
- 使用"紧张氛围"代替"恐怖场景"
- 描述场景而非具体人物特征

### 4. 成本控制

**优化建议**:
- 只为重要章节生成插图
- 复用相似场景的插图
- 使用缓存避免重复生成
- 设置合理的配额限制

```http
POST https://dashscope.aliyuncs.com/api/v1/services/aigc/text2image/image-synthesis
```
