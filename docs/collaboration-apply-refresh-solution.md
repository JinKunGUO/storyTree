**状态**：✅ 已完成

## 📊 问题回顾

### 原始问题
用户点击"申请协作"后，如果不刷新页面：
- ❌ 点击故事章节时，只显示"AI续写"按钮
- ❌ **人工续写按钮不显示**
- ❌ 用户体验差，需要手动刷新

### 根本原因
续写按钮的显示依赖于实时权限检查（`/api/stories/:id/role`），申请协作后如果不刷新页面，权限状态不会更新，导致续写按钮不显示。

## 🔧 实施内容

### 修改文件
- **文件**：`web/story.html`
- **函数**：`submitCollaborationRequest`
- **位置**：第2464-2476行

### 修改前代码

```javascript
const data = await response.json();

if (data.auto_approved) {
    showSuccess('申请已自动通过！现在你是这个故事的协作者了');
    setTimeout(() => window.location.reload(), 1500);
} else {
    showSuccess('申请已提交，请等待主创审核');
    closeApplyCollaborationModal();
    
    // 重新检查用户角色，更新按钮状态

        checkIfAuthor(window.currentStory);
    }, 500);

```

**问题**：
- 自动通过时：刷新页面 ✅
- 需要审核时：不刷新页面，只调用 `checkIfAuthor()` ❌
- `checkIfAuthor()` 只更新顶部按钮，不更新节点详情面板中的续写按钮

### 修改后代码

```javascript
const data = await response.json();

if (data.auto_approved) {
    showSuccess('申请已自动通过！现在你是这个故事的协作者了，页面即将刷新');
} else {
    showSuccess('申请已提交，请等待主创审核，页面即将刷新');
}

// ✅ 统一处理：无论自动通过还是需要审核，都刷新页面
// 这样确保所有按钮状态（包括续写按钮）都能正确更新
setTimeout(() => {
    window.location.reload();
}, 1500);
```

**改进**：
- ✅ 自动通过时：刷新页面
- ✅ 需要审核时：也刷新页面
- ✅ 确保所有按钮状态（包括续写按钮）都能正确更新

## 📈 技术细节

### 续写按钮显示逻辑

**位置**：`web/story.html:4362-4403`

```javascript
// 检查用户权限，决定是否显示"续写分支"按钮
const roleResponse = await fetch(`/api/stories/${storyId}/role`, {
    headers: { 'Authorization': `Bearer ${token}` }
});

if (roleResponse.ok) {
    const roleData = await roleResponse.json();
    
    // 只有作者或有效协作者才能续写
    if (roleData.is_author) {
        continueBtn.style.display = 'block';
    } else if (roleData.is_collaborator && roleData.allow_branch) {
        continueBtn.style.display = 'block';
    } else {
        continueBtn.style.display = 'none';
    }
}
```

**关键点**：
- 续写按钮的显示依赖于 `/api/stories/:id/role` 接口
- 这个接口返回用户的角色信息（`is_author`, `is_collaborator`, `allow_branch`）
- 页面刷新时，会重新请求这个接口，获取最新的角色信息

### 权限检查时机

| 时机 | 触发场景 | 结果 |
|------|---------|------|
| 页面加载时 | 用户访问故事页面 | 获取用户角色 |
| 点击节点时 | 用户点击章节节点 | 检查续写权限 |
| 申请协作后 | 用户提交协作申请 | **刷新页面，重新获取角色** |
| 退出协作后 | 用户退出协作 | 刷新页面，重新获取角色 |
