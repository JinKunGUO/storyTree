# AI插图配额修复说明

## 📅 修复日期
2026-03-06

## 🐛 问题描述

用户在使用AI插图生成功能时遇到以下错误：

```
POST http://localhost:3001/api/ai/v2/illustration/submit 403 (Forbidden)
生成插图失败: Error: 本月AI插图次数已用完（0/0）
```

### 根本原因

**等级1（新手作者）的AI插图配额设置为0**，导致新手用户无法使用插图功能。

```typescript
// 修复前
1: {
  name: '新手作者',
  minPoints: 0,
  maxPoints: 99,
  quotas: {
    continuation: 3,
    polish: 5,
    illustration: 0   // ❌ 配额为0
  }
}
```

### 业务逻辑

根据 `api/src/utils/points.ts` 的 `canUseAiFeature` 函数：

1. 首先检查月度配额是否用完
2. 如果配额为0，则 `used(0) >= limit(0)` 返回true
3. 返回错误：`本月AI插图次数已用完（0/0）`

## ✅ 修复方案

### 1. 调整等级1的插图配额

将等级1的AI插图配额从 **0改为2**，让新手用户也能体验插图功能。

```typescript
// 修复后
1: {
  name: '新手作者',
  minPoints: 0,
  maxPoints: 99,
  quotas: {
    continuation: 3,  // 每月AI续写次数
    polish: 5,        // 每月AI润色次数
    illustration: 2   // ✅ 每月AI插图张数（从0改为2）
  }
}
```

### 2. 完整的等级配额表

| 等级 | 名称 | 积分范围 | AI续写 | AI润色 | AI插图 |
|------|------|----------|--------|--------|--------|
| Lv1 | 新手作者 | 0-99 | 3次/月 | 5次/月 | **2张/月** ✅ |
| Lv2 | 活跃作者 | 100-499 | 10次/月 | 20次/月 | 2张/月 |
| Lv3 | 专业作者 | 500-1999 | 30次/月 | 无限 | 10张/月 |
| Lv4 | 大师作者 | 2000+ | 无限 | 无限 | 50张/月 |

### 3. 修改的文件

- `api/src/utils/points.ts` - 等级配额配置

## 💡 设计考虑

### 为什么给新手用户2张配额？

1. **体验核心功能**
   - 插图是重要的增值功能
   - 新手用户需要体验才能感受价值
   - 2张配额足够测试和体验

2. **鼓励升级**
   - 体验后如果喜欢，会想要更多配额
   - 激励用户升级等级或购买会员
   - 平衡免费体验和商业价值

3. **成本控制**
   - AI插图生成成本较高（DALL-E 3）
   - 2张配额是合理的免费额度
   - 不会造成过大的成本压力

4. **用户留存**
   - 完全禁用会导致新手流失
   - 少量配额能提升用户粘性
   - 增加用户活跃度

## 📊 配额消耗规则

### 积分消耗

| 功能 | 积分消耗 |
|------|----------|
| AI续写 | 10积分 |
| AI润色 | 3积分 |
| AI插图 | 20积分 |

### 配额使用逻辑

```typescript
// 配额检查逻辑
if (月度配额未用完) {
  ✅ 允许使用，消耗配额
} else if (有足够积分) {
  ✅ 允许使用，消耗积分
} else {
  ❌ 拒绝使用，提示升级或充值
}
```

### 配额重置

- 每月1号00:00自动重置
- 重置后 `used = 0`
- 配额不会累积到下个月
