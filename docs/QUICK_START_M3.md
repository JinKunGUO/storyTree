# 🚀 StoryTree M3 新功能快速体验指南

## 欢迎体验 StoryTree v1.0.11！

本指南将带你快速体验M3阶段新增的所有功能。

---

## 📋 准备工作

### 1. 启动服务
```bash
# 进入项目目录
cd /Users/jinkun/storytree

# 启动后端服务
cd api && npm run dev
```

### 2. 访问应用
打开浏览器访问：`http://localhost:3001`

---

## 🎯 功能体验流程

### 第一步：注册/登录 👤

1. 访问首页 `http://localhost:3001`
2. 点击右上角"注册"按钮
3. 填写用户信息并注册
4. 或者直接登录已有账号

**提示**: 注册后会自动登录并跳转到首页

---

### 第二步：发现故事 🔍

1. 点击导航栏的"发现故事"或访问 `http://localhost:3001/discover`
2. 浏览所有公开的故事
3. 使用搜索框搜索感兴趣的故事
4. 使用筛选标签（全部/热门/最新/已完结）

**新功能**: 
- ✨ 点击任意故事卡片，跳转到故事详情页

---

### 第三步：查看故事详情 📖

当你点击一个故事后，会进入故事详情页面：

**页面URL**: `http://localhost:3001/story?id={故事ID}`

**你可以看到**:
- 📚 故事封面和基本信息
- 📊 统计数据（阅读量、点赞数、章节数、字数）
- 📝 故事简介
- 👤 作者信息
- 📑 完整的章节列表

**功能体验**:
1. **章节排序**: 点击"正序"/"倒序"按钮切换章节顺序
2. **开始阅读**: 点击"开始阅读"按钮跳转到第一章
3. **点击章节**: 点击任意章节卡片进入阅读页面
4. **关注作者**: 点击"关注"按钮（功能开发中）

---

### 第四步：阅读章节 📕

点击章节后进入沉浸式阅读页面：

**页面URL**: `http://localhost:3001/chapter?id={章节ID}`

**阅读界面特点**:
- 🎨 清爽的白色阅读区
- 📏 舒适的行高和字体
- 🔧 顶部固定工具栏
- 💬 底部评论区

**功能体验**:
1. **返回故事**: 点击左上角的返回箭头
2. **章节导航**: 使用"上一章"/"下一章"按钮
3. **阅读设置**: 点击齿轮图标调整字体大小
   - 小号字体（16px）
   - 中号字体（18px）- 默认
   - 大号字体（20px）
4. **书签功能**: 点击书签图标（开发中）

---

### 第五步：发表评论 💬

在章节阅读页面底部，你可以参与讨论：

**评论功能**:
1. **发表评论**:
   - 在评论输入框输入内容（最多500字）
   - 实时显示字符计数
   - 点击"发表评论"按钮提交

2. **回复评论**:
   - 点击任意评论下的"回复"按钮
   - 在弹出的回复框中输入内容
   - 点击"发表回复"提交

3. **管理评论**:
   - 编辑自己的评论（点击"编辑"）
   - 删除自己的评论（点击"删除"）
   - 管理员可以删除任意评论

4. **浏览评论**:
   - 树形结构显示评论和回复
   - 自动分页加载（每页20条）
   - 智能时间显示（刚刚、X分钟前等）

## 🐛 遇到问题？

### 常见问题

**Q: 点击故事卡片没反应？**
A: 请确保后端服务正在运行（`npm run dev`）

**Q: 评论发表失败？**
A: 请确保已登录，且评论内容不为空

**Q: 页面显示空白？**
A: 请检查浏览器控制台是否有错误信息

**Q: 章节导航按钮被禁用？**
A: 如果是第一章，"上一章"会被禁用；如果是最后一章，"下一章"会被禁用

### 技术支持

如遇到其他问题，请查看：
- 浏览器控制台错误信息
- 后端服务日志
- `docs/M3_FEATURE_TEST.md` 测试文档
