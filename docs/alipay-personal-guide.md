# 支付宝个人当面付申请与配置指南

## 📋 一、快速概览

**适合人群**：个人开发者、学生、自由职业者、小微商户

**申请条件**：
- ✅ 实名认证的支付宝个人账号
- ✅ 中国大陆居民身份证
- ✅ 年满 18 周岁

**费率**：0.6%（与企业相同）

**限额**：
- 单笔限额：5000 元
- 日累计限额：5000-10000 元（根据信用评级）
- 月累计限额：约 10 万元

**到账时间**：实时到账支付宝余额

**提现**：免费，直接提现到银行卡

## 🚀 二、申请流程（30 分钟完成）

### 步骤 1：申请当面付功能（10 分钟）

1. **打开支付宝 APP**
   
2. **搜索「商家服务」**
   - 在支付宝首页搜索框输入「商家服务」
   - 点击进入官方小程序

3. **点击「收钱码」**
   - 在商家服务页面找到「收钱码」选项
   - 点击进入

4. **申请「当面付」**
   - 找到「当面付」功能
   - 点击「立即申请」
   - 阅读并同意协议

5. **填写个人信息**
   ```
   - 姓名：（与支付宝实名认证一致）
   - 身份证号：（18 位身份证号码）
   - 手机号：（接收验证码）
   ```

6. **提交审核**
   - 确认信息无误
   - 提交申请
   - 等待审核（通常 1-3 个工作日）

**审核结果通知**：
- 审核通过会收到短信通知
- 在「商家服务」中可以查看申请进度

---

### 步骤 2：创建应用获取密钥（20 分钟）

#### 2.1 访问支付宝开放平台

```
https://open.alipay.com/
```

#### 2.2 登录

- 点击页面右上角「登录」
- 使用个人支付宝账号**扫码登录**
- 完成登录

#### 2.3 进入开发者中心

- 登录后，点击顶部菜单「开发者中心」
- 选择「网页/移动应用」

#### 2.4 创建应用

1. **点击「创建应用」**

2. **填写应用信息**：
   ```
   应用类型：其他（选择「其他」类型）
   应用名称：StoryTree
   应用图标：上传一个 Logo（可选，可先用默认图标）
   应用描述：AI 故事创作平台
   ```

3. **点击「确定」**创建应用

#### 2.5 添加产品

1. **在应用详情页面**，找到「产品管理」
2. **点击「添加产品」**
3. **选择「当面付」**
   - 在产品列表中找到「当面付」
   - 点击「添加」
4. **签约**
   - 点击「签约」按钮
   - 阅读并同意协议
   - 提交签约申请

**注意**：签约可能需要审核，审核通过后才能使用。

#### 2.6 生成密钥对

打开终端，执行以下命令生成 RSA 密钥对：

```bash
# 进入工作目录
cd /Users/jinkun/storytree

# 创建密钥文件夹
mkdir -p alipay_keys
cd alipay_keys

# 生成私钥（2048 位 RSA）
openssl genrsa -out app_private_key.pem 2048

# 生成公钥（从私钥提取）
openssl rsa -in app_private_key.pem -pubout -out app_public_key.pem

# 查看生成的文件
ls -la

# 查看私钥内容
echo "=== 应用私钥 ==="
cat app_private_key.pem

echo ""
echo "=== 应用公钥 ==="
cat app_public_key.pem
```

**密钥文件说明**：
- `app_private_key.pem` - 应用私钥（自己保存，不要泄露）
- `app_public_key.pem` - 应用公钥（上传到支付宝）

#### 2.7 上传公钥到支付宝

1. **回到应用详情页面**

2. **点击「开发信息」**
   - 在左侧菜单找到「开发信息」
   - 点击进入

3. **配置接口加签方式**
   - 找到「接口加签方式」
   - 点击「配置」或「修改」
   - 选择「公钥」模式（**注意：不是证书模式**）

4. **上传应用公钥**
   - 打开 `app_public_key.pem` 文件
   - 复制文件内容（**去掉**首尾的 `-----BEGIN PUBLIC KEY-----` 和 `-----END PUBLIC KEY-----` 标记）
   - 粘贴到「应用公钥」输入框
   - 点击「保存」

5. **获取支付宝公钥**
   - 保存后，系统会生成**支付宝公钥**
   - 点击「查看」或「下载」
   - 复制并妥善保存支付宝公钥

**重要提示**：
- 应用私钥：自己保存，用于签名请求
- 支付宝公钥：用于验证回调签名

### 步骤 3：配置环境变量（5 分钟）

#### 3.1 整理密钥格式

支付宝配置需要将密钥**去掉 PEM 标记，合并成一行**。

使用以下命令快速处理：

```bash
cd /Users/jinkun/storytree/alipay_keys

# 提取私钥（去掉 PEM 标记，合并成一行）
echo "=== 应用私钥（已格式化） ==="
grep -v "BEGIN\|END" app_private_key.pem | tr -d '\n'
echo ""

# 提取支付宝公钥（假设您已保存到文件）
# 如果没有，手动从支付宝开放平台复制
```

#### 3.2 编辑 .env 文件

打开 `api/.env` 文件，填入配置：

```env
# ===================================
# 支付宝支付配置（个人当面付）
# ===================================

# 支付宝应用 ID（从应用详情页面获取）
ALIPAY_APP_ID=2021001234567890

# 支付宝网关地址
ALIPAY_GATEWAY=https://openapi.alipay.com/gateway.do

# 应用私钥（格式化后的私钥，一行）
ALIPAY_PRIVATE_KEY=MIIEvQIBADANBgkqhkiG9w0BAQEFAASC...（完整的一行）

# 支付宝公钥（格式化后的公钥，一行）
ALIPAY_PUBLIC_KEY=MIIBIjANBgkqhkiG9w0BAQEFAAOCAQ8A...（完整的一行）

# 支付回调通知地址（开发环境）
# 如需测试回调，使用 ngrok 暴露到公网
ALIPAY_CALLBACK_URL=http://localhost:3001

# API 基础 URL（用于生成回调地址）
API_URL=http://localhost:3001
```

**配置说明**：
- `ALIPAY_APP_ID`：应用详情页面可见
- `ALIPAY_PRIVATE_KEY`：应用私钥（去掉 PEM 标记）
- `ALIPAY_PUBLIC_KEY`：支付宝公钥（去掉 PEM 标记）

### 3.2 使用 ngrok 测试回调（可选）

支付宝回调需要公网地址，开发环境可以使用 ngrok：

```bash
# 安装 ngrok（如果未安装）
brew install ngrok

# 注册 ngrok 获取 authtoken
# 访问：https://ngrok.com/

# 配置 authtoken
ngrok config add-authtoken YOUR_AUTHTOKEN

# 启动内网穿透
ngrok http 3001
```

ngrok 会生成一个公网地址，如：
```
https://abc123.ngrok.io
```

将此地址配置到 `.env` 文件：

```env
ALIPAY_CALLBACK_URL=https://abc123.ngrok.io
API_URL=https://abc123.ngrok.io
```

**注意**：
- ngrok 免费版每次重启地址会变
- 建议购买付费版或使用其他固定域名工具
- 也可以使用国内的 natapp、神卓互联等

## 🐛 六、常见问题

### Q1: 个人当面付和企业当面付有什么区别？

**答**：
| 对比项 | 个人 | 企业 |
|--------|------|------|
| 费率 | 0.6% | 0.6% |
| 单笔限额 | 5000 元 | 可协商 |
| 日限额 | 5000-10000 元 | 可协商 |
| 申请要求 | 身份证 | 营业执照 |
| 资金到账 | 支付宝余额 | 对公账户 |
| 提现 | 免费 | 免费 |

### Q2: 申请被拒绝怎么办？

**答**：
1. 检查是否满足申请条件（实名认证、年满 18 岁）
2. 检查支付宝账号是否正常使用
3. 等待 7 天后重新申请
4. 联系支付宝客服：95188

### Q3: 密钥格式不对怎么办？

**答**：
密钥必须是**去掉 PEM 标记，合并成一行**的格式。

错误示例（有 PEM 标记）：
```
-----BEGIN RSA PRIVATE KEY-----
MIIEpAIBAAKCAQEA...
-----END RSA PRIVATE KEY-----
```

正确示例（去掉标记，一行）：
```
MIIEpAIBAAKCAQEA...（完整的一行）
```

使用命令快速格式化：
```bash
grep -v "BEGIN\|END" app_private_key.pem | tr -d '\n'
```

### Q4: 回调地址配置后收不到通知？

**答**：
1. 检查回调地址是否可公网访问
2. 使用 ngrok 等工具暴露本地服务
3. 检查防火墙设置
4. 查看服务器日志
5. 在支付宝开放平台测试回调地址

### Q5: 交易限额不够用怎么办？

**答**：
1. 提升信用评级（多使用支付宝）
2. 申请成为个体户，升级为企业支付宝
3. 使用多个账号分流（不推荐）
4. 引导用户使用其他方式（如银行转账）

### Q6: 支持自动续费吗？

**答**：
个人当面付**不支持**自动续费功能。

自动续费需要：
- 企业支付宝账号
- 签约「周期性扣款」产品

**替代方案**：
- 会员到期前发送提醒
- 用户手动续费
- 提供续费优惠
