# 📝 故事编辑快速参考

## 🎯 三种操作方式

### 1️⃣ 编辑故事基本信息

**在哪里**: 故事详情页 `http://localhost:3001/story?id=7`  
**按钮**: "编辑故事" (紫色按钮)  
**可编辑**:
- ✅ 故事标题
- ✅ 故事简介
- ✅ 故事封面

### 2️⃣ 续写新章节 ⭐ 新功能

**在哪里**: 故事详情页 `http://localhost:3001/story?id=7`  
**按钮**: "续写章节" (紫色按钮，✏️ 图标)  
**可以做**:
- ✅ 撰写新章节
- ✅ 添加章节标题和内容
- ✅ 使用富文本编辑器
- ✅ 发布或保存草稿

### 3️⃣ 编辑已有章节

**在哪里**: 章节阅读页 `http://localhost:3001/chapter?id=1`  
**按钮**: ✏️ 编辑按钮 (右上角工具栏)  
**可编辑**:
- ✅ 章节标题
- ✅ 章节正文内容

---

## ⚡ 快速操作流程

```
编辑故事信息:
http://localhost:3001/story?id=7 → [编辑故事] → 修改 → 保存

续写新章节: ⭐ 新功能
http://localhost:3001/story?id=7 → [续写章节] → 撰写 → 发布

编辑已有章节:
http://localhost:3001/story?id=7 → 点击章节 → [✏️] → 修改 → 保存
```

---

## 🔐 权限要求

- ✅ 必须登录
- ✅ 必须是故事/章节的作者
- ❌ 其他用户看不到编辑按钮
