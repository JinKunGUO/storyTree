```javascript:1536:1606:web/chapter.html
async function loadBreadcrumbNav(nodeId) {
    const response = await fetch(`/api/stories/nodes/${nodeId}/path`);
    const data = await response.json();
    
    // 渲染面包屑
    let breadcrumbHTML = '';
    
    // 添加故事标题
    if (data.story) {
        breadcrumbHTML += `
            <a href="/story?id=${data.story.id}" class="breadcrumb-item">
                <i class="fas fa-book"></i>
                <span>${data.story.title}</span>
            </a>
        `;
    }
    
    // 添加路径节点
    data.path.forEach((node, index) => {
        if (index === data.path.length - 1) {
            // 当前章节
            breadcrumbHTML += `
                <span class="breadcrumb-item current">
                    <i class="fas fa-file-alt"></i>
                    <span>${node.title}</span>
                </span>
            `;
        } else {
            // 可点击的历史节点
            breadcrumbHTML += `
                <a href="/chapter?id=${node.id}" class="breadcrumb-item">
                    <i class="fas fa-file-alt"></i>
                    <span>${node.title}</span>
                </a>
            `;
        }
    });
}
```

**代码示例：**
```typescript:371:421:api/src/routes/nodes.ts
router.get('/:id/siblings', async (req, res) => {
  const { id } = req.params;
  
  try {
    const node = await prisma.nodes.findUnique({
      where: { id: parseInt(id) },
      select: { parent_id: true }
    });
    
    if (!node || !node.parent_id) {
      return res.json({ siblings: [] });
    }
    
    const siblings = await prisma.nodes.findMany({
      where: {
        parent_id: node.parent_id,
        id: { not: parseInt(id) }
      },
      include: {
        author: { select: { id: true, username: true } },
        _count: { select: { comments: true, ratings: true } }
      },
      orderBy: [
        { rating_avg: 'desc' },
        { read_count: 'desc' }
      ]
    });
    
    res.json({ siblings });
  } catch (error) {
    res.status(500).json({ error: 'Failed to fetch siblings' });
  }
});
```

**功能说明：**

- ✅ 展示推荐理由和统计数据
- ✅ 显示备选分支列表
- ✅ 智能显示：只在有子分支时显示推荐按钮
- ✅ 精美的AI推荐界面

**核心功能：**
1. **分支评分算法**
   - 阅读量权重：40%
   - 评分权重：40%
   - 评论数权重：20%
   - 对数归一化处理

2. **AI推荐API**
   - 端点：`POST /api/ai/v2/recommend-branch`
   - 分析所有子分支质量
   - 返回推荐分支和备选方案
   - 生成推荐理由

3. **推荐界面**
   - AI推荐按钮（魔法棒图标）
   - 推荐理由卡片
   - 推荐分支详情展示
   - 备选分支列表
   - 一键跳转阅读

**评分算法示例：**
```typescript
const calculateBranchScore = (branch: any) => {
  const readWeight = 0.4;      // 阅读量权重
  const ratingWeight = 0.4;    // 评分权重
  const commentWeight = 0.2;   // 评论数权重

  const readScore = Math.log(branch.read_count + 1) * readWeight * 10;
  const ratingScore = (branch.rating_avg || 0) * ratingWeight * 10;
  const commentScore = Math.log(branch._count.comments + 1) * commentWeight * 10;

  return readScore + ratingScore + commentScore;
};
```

**推荐理由生成：**
- 高阅读量（>50次）→ "高阅读量(N次)"
- 高评分（>=4.0分）→ "高评分(N分)"
- 活跃讨论（>5条评论）→ "活跃讨论(N条评论)"
- 综合数据分析
