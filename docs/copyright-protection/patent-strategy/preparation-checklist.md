# 专利申请材料准备清单

> 版本：v1.0 | 更新日期：2026-04-07

---

## 一、申请前必须准备的材料

### 1.1 申请人资质材料

- [ ] **营业执照副本**（加盖公章）
- [ ] **组织机构代码证**（若已三证合一则只需营业执照）
- [ ] **法定代表人身份证**（正反面）
- [ ] **专利申请授权委托书**（委托代理机构时需要，代理机构提供模板）
- [ ] **公司银行账户信息**（用于退款等）

### 1.2 发明人信息

每位发明人需提供：
- [ ] 姓名（与身份证一致）
- [ ] 身份证号码
- [ ] 工作单位
- [ ] 联系方式
- [ ] 与公司的关系（员工/顾问等）

**注意**：职务发明（员工在职期间完成的发明）归公司所有，需在发明人信息中注明。

## 二、技术交底书（核心材料）

技术交底书是专利申请的核心文件，由技术团队撰写，提交给专利代理机构。

### 2.1 技术交底书模板

```markdown
# 技术交底书

## 基本信息
- 发明名称：[具体技术名称]
- 发明人：[姓名列表]
- 撰写日期：[日期]
- 技术领域：计算机软件 / 互联网

## 一、技术背景
### 现有技术描述
[描述目前市场上类似技术的现状]

### 现有技术的缺点
[指出现有技术存在的问题，为本发明的创新性做铺垫]

## 二、发明内容
### 发明目的
[本发明要解决的技术问题]

### 技术方案
[详细描述技术实现方案，包括：
- 系统架构
- 核心算法流程
- 关键数据结构
- 各模块功能]

### 有益效果
[本发明相比现有技术的优势]

## 三、具体实施方式
### 实施例一
[最典型的实现方式，配合流程图说明]

### 实施例二（可选）
[其他实现变体]

## 四、附图说明
[列出所有附图及其说明]

## 五、权利要求建议（可选）
[如果有想法，可以列出希望保护的核心权利要求]
```

### 2.2 技术交底书要点

**必须包含的内容**：
- [ ] 技术背景（现有技术及其不足）
- [ ] 发明要解决的技术问题
- [ ] 技术方案的完整描述
- [ ] 系统架构图（手绘或电子图均可）
- [ ] 核心算法流程图
- [ ] 实施例（至少1个）
- [ ] 技术效果说明

**注意事项**：
- 描述要具体，避免过于抽象
- 区分"必要技术特征"（权利要求中的）和"优选特征"
- 如有代码，提供关键代码片段（不需要全部）
- 附图要清晰，标注要完整

## 三、各专利项目技术交底书准备指引

### 3.1 分支叙事引擎专利

**需要准备的内容**：

- [ ] 故事树数据结构的详细设计（ER图或JSON示例）
- [ ] 节点创建、查询、遍历的算法描述
- [ ] 性能优化方案（索引设计、缓存策略）
- [ ] 与现有线性叙事系统的对比

**关键附图**：
- 故事树数据结构示意图
- 节点查询算法流程图
- 系统架构图

### 3.2 AI 协作创作专利

**需要准备的内容**：

- [ ] 上下文提取算法的完整描述
- [ ] 角色特征向量的定义和提取方法
- [ ] 风格特征的量化方法
- [ ] AI生成内容的后处理流程
- [ ] 与通用AI续写的效果对比

**关键附图**：
- AI续写系统架构图
- 上下文提取流程图
- 角色一致性检测流程图

### 3.3 版权管理系统专利

**关键附图**：
- 版权管理系统架构图

## 四、申请流程检查清单

### 4.1 提交前检查

- [ ] 技术交底书已由技术负责人审核
- [ ] 确认技术内容未在申请日前公开披露
- [ ] 发明人信息已确认（所有参与人均已列入）
- [ ] 已与代理机构签署委托协议
- [ ] 代理费用已支付或已确认付款安排
- [ ] 申请授权委托书已加盖公章

### 4.2 代理机构提交后跟进

- [ ] 收到申请受理通知书（含申请号）
- [ ] 申请号已记录在内部知识产权台账
- [ ] 设置审查意见答复提醒（通常提交后12-18个月）
- [ ] 通知发明人申请已提交

### 4.3 审查阶段

- [ ] 收到审查意见通知书（如有）
- [ ] 在规定期限内（通常2-4个月）提交答复
- [ ] 答复前与代理机构充分沟通修改策略
- [ ] 答复后跟进审查结果

## 六、重要联系信息

### 6.1 国家知识产权局

- 官网：https://www.cnipa.gov.cn/
- 专利查询：https://pss-system.cponline.cnipa.gov.cn/
- 在线申请：https://cponline.cnipa.gov.cn/
- 咨询电话：010-62083114

### 6.2 推荐专利代理机构类型

选择代理机构时，重点考察：
- 是否有互联网/AI/软件领域的专利申请经验
- 代理人的技术背景（计算机相关专业优先）
- 历史授权率
- 服务响应速度

### 6.3 专利相关法律法规

- 《中华人民共和国专利法》（2020年修订）
- 《专利法实施细则》
- 《专利审查指南》（软件专利申请参考第二部分第九章）
