## 核心问题与解答

### Q: StoryTree 最大的版权风险是什么？
多人协作创作的版权归属问题。当多位创作者共同构建一个故事树时，各章节的版权归属、收益分配、修改权限等均需要明确的法律约定。详见 [创作协议](./user-agreement/creation-agreement.md)。

### Q: 区块链存证能解决什么问题？
提供不可篡改的创作时间戳证明，在版权纠纷中作为证据使用。详见 [区块链存证方案](./blockchain-certification/antchain-guide.md)。

### Q: StoryTree 有哪些技术可以申请专利？
分支叙事引擎、AI协作创作算法、读者参与度量化模型等。详见 [可申请专利技术点分析](./patent-strategy/patentable-technologies.md)。
