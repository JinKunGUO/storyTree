## 👤 快速登录

### 开发环境快速登录
StoryTree在开发环境提供了快速登录功能，无需注册即可体验：

1. 点击右上角"登录"按钮
2. 输入任意用户名（如：`testuser`）
3. 输入任意密码（如：`123456`）
4. 点击登录即可

**提示**: 
- 首次登录会自动创建账户
- 开发环境不验证密码强度
- 建议使用不同用户名体验多用户功能

### 创建管理员账户
如需体验管理功能，需要设置管理员权限：

```bash
# 进入API目录
cd /Users/jinkun/storytree/api

# 使用Prisma Studio
npx prisma studio

# 在浏览器中打开 http://localhost:5555
# 找到User表，编辑用户，将isAdmin设为true
```

或使用SQL直接设置：
```bash
# SQLite
cd /Users/jinkun/storytree/api
sqlite3 prisma/dev.db
UPDATE users SET is_admin = 1 WHERE username = 'testuser';
.quit
```

## 🎮 功能体验指南

### 1. 📖 创建你的第一个故事

**步骤**：
1. 登录后，点击导航栏"创作"
2. 填写故事信息：
   - 故事标题：如"时空旅行者"
   - 故事简介：简短描述你的故事
   - 封面图片：点击上传图片（可选）
3. 填写第一章：
   - 章节标题：如"第一章：神秘的邀请函"
   - 章节内容：开始你的创作
   - 配图：为章节添加图片（可选）
4. 点击"发布故事"

**提示**：
- 内容会自动进行审核检查
- 新用户前3篇内容需要人工审核
- 避免使用敏感词汇

### 2. 🌿 创建分支剧情

**体验分支创作的魅力**：
1. 在首页浏览故事列表
2. 点击进入任意故事
3. 阅读现有章节
4. 在章节底部点击"创建分支"
5. 编写你的续写：
   - 章节标题
   - 章节内容
   - 可添加配图
6. 发布分支

**效果**：
- 故事会形成树状结构
- 读者可以选择不同的剧情走向
- 每个分支都是独立的故事线

### 3. 🤖 使用AI续写

**让AI帮你创作**：
1. 进入任意章节
2. 点击"AI续写"按钮
3. 查看AI生成的3个续写选项：
   - 🎭 戏剧性：情节跌宕起伏
   - 💭 哲理性：深度思考和反思
   - 🌸 温馨：温暖治愈的故事
4. 选择喜欢的版本
5. 可以编辑后再发布

**提示**：
- AI续写基于当前章节内容
- 可以多次生成不同选项
- 建议在AI基础上加入个人创意

### 4. ⭐ 评分和互动

**参与社区互动**：
1. 阅读章节后可以评分（1-5星）
2. 高分章节会优先展示
3. 查看章节的平均评分和阅读数
4. 通过评分引导最佳剧情

### 5. 🔍 搜索功能

**快速找到感兴趣的内容**：
1. 点击导航栏"🔍 搜索"
2. 输入关键词（如"科幻"、"爱情"）
3. 查看搜索结果：
   - 故事列表
   - 章节列表
4. 点击结果快速跳转

**搜索技巧**：
- 支持标题和内容搜索
- 只搜索已审核通过的内容
- 结果按相关度排序

### 6. 👥 社交功能

**关注作者，获取动态**：

#### 查看用户资料
1. 点击任意作者名字
2. 查看个人主页：
   - 创作统计（故事数、章节数）
   - 社交数据（关注、粉丝）
   - 创作历史
   - 参与的章节

#### 关注作者
1. 在用户主页点击"关注"
2. 关注后可以：
   - 在动态流中看到TA的新作品
   - TA会收到关注通知

#### 查看动态流
1. 点击导航栏"动态"
2. 查看关注用户的最新创作
3. 快速访问感兴趣的内容

#### 编辑个人资料
1. 点击右上角用户名
2. 选择"编辑资料"
3. 可以修改：
   - 个人简介
   - 头像（上传图片）

### 7. 🔔 通知系统

**及时了解互动信息**：

#### 查看通知
1. 点击导航栏"🔔"图标
2. 红色数字显示未读数量
3. 查看通知列表

#### 通知类型
- **关注通知**：有人关注了你
- **审核通知**：内容审核结果
- **更新通知**：关注的故事有更新
- **系统通知**：重要系统消息

#### 管理通知
- 点击通知跳转到相关内容
- 标记单条已读
- 全部标记已读

### 8. 📸 图片功能

**让故事更生动**：

#### 上传封面
1. 创建故事时点击"上传封面"
2. 选择图片文件
3. 支持格式：jpg, png, gif, webp
4. 最大5MB

#### 添加配图
1. 创建章节时点击"上传配图"
2. 为章节添加插图
3. 图片会显示在章节内容中

#### 设置头像
1. 进入个人资料编辑
2. 点击"上传头像"
3. 选择个人头像图片

### 9. 🛡️ 管理员功能

**内容审核和管理**（需要管理员权限）：

#### 访问管理后台
```bash
# 在浏览器中打开
open http://localhost:3001/admin.html
```

#### 审核队列
1. 查看待审核内容列表
2. 查看内容详情
3. 进行审核操作：
   - ✅ 通过：内容正常，批准发布
   - ❌ 驳回：内容不合规，拒绝发布
   - 🚫 下架：已发布内容违规，下架处理

#### 举报处理
1. 查看用户举报的内容
2. 了解举报原因
3. 根据情况处理

#### 审核通知
- 审核结果会自动通知作者
- 可以添加审核备注

## 🐛 常见问题

### Q1: 登录后没有反应？
**A**: 检查浏览器控制台是否有错误，确保后端服务正常运行。

### Q2: 图片上传失败？
**A**: 
- 检查图片大小是否超过5MB
- 确认图片格式是否支持
- 查看uploads目录是否有写入权限

### Q3: 搜索没有结果？
**A**: 
- 确保数据库中有已审核通过的内容
- 尝试更换搜索关键词
- 搜索词至少需要2个字符

### Q4: AI续写不工作？
**A**: 
- 检查是否配置了AI API密钥
- 查看后端日志是否有错误
- 确认网络连接正常

### Q5: 通知不显示？
**A**: 
- 刷新页面重新加载通知
- 检查是否已登录
- 确认是否有新的互动

### Q6: 如何成为管理员？
**A**: 使用数据库工具或SQL命令设置isAdmin字段为true。
