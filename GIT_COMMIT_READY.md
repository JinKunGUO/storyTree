## 🚀 如何提交

### 方法1: 使用提交脚本（推荐）

```bash
# 执行提交脚本
./commit-ai-illustration.sh
```

脚本会：
1. 显示修改的文件列表和统计
2. 显示完整的提交信息
3. 询问确认后提交到本地仓库
4. 询问是否推送到远程仓库

### 方法2: 手动提交

```bash
# 1. 查看状态（已经添加到暂存区）
git status

# 2. 提交到本地仓库
git commit -m "feat: AI插图生成功能完整实现 - 千问通义万相集成

详细提交信息请查看: docs/2026-03-06-AI插图功能完整实现总结.md
"

# 3. 推送到远程仓库
git push origin feature/m3-user-auth
```
